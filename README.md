## Setup
```
$ git clone https://lab.ssafy.com/s05-webmobile3-sub3/S05P13A109.git
$ cd S05P13A109
$ npm install

// backend
$ cd ./S05P13A109/backend/src
$ npm install

// frontend
$ cd ./S05P13A109/frontend
$ npm install
```

### Backend
- Node.js
- Express
- JWT (JSON Web Tokens)
- AWS (Amazon Web Service)
- MariaDB
- Docker
- Nginx
- Bycrpt

- React.js
- Electron
- Material-ui

- GCP (Google Cloud Platform)
- OpenCV
